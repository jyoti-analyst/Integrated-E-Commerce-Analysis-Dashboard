# Integrated-E-Commerce-Analysis-Dashboard
This project showcases an interactive E-Commerce Analysis Dashboard built using Tableau. It focuses on analyzing key business areas such as marketing performance, operational efficiency, and customer behaviour to support data-driven decision-making.
# 🎯 Objectives
Analyze overall sales performance and ROI
Evaluate marketing channel effectiveness
Monitor operational metrics like return rate and delivery delays
Understand customer behavior (new vs repeat customers)
Identify category-level performance trends
# 🛠️ Tools & Technologies
Tableau Public
Data Cleaning & Preparation
# 📊 Key Features
KPI tracking: Total Sales, ROI, Conversion Rate
Channel-wise performance (Google, Email, Facebook)
Return rate and delivery delay analysis
Customer segmentation (New vs Repeat)
Category & sub-category level insights
Interactive filters for dynamic analysis
# 🔍 Key Insights
Google drives the highest sales and profit among all channels
Conversion rate (~3%) indicates scope for funnel optimization
Return rate (~15%) highlights potential improvement areas
Balanced contribution from new and repeat customers
Regional performance is mostly consistent with minor variations
# 💡 Skills Demonstrated
Data Visualization & Dashboard Design
Business Insight Generation
Data Storytelling
